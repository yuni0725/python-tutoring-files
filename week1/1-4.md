#### 문제

다음 규칙을 따라 사이트별로 비밀번호를 만들어주는 프로그램을 작성하시오.

예시 : https://google.com(단, 입력값은 https:// 로 시작하고 .com으로 끝난다.)

규칙1 : https:// 부분은 제외한다 -> google.com

규칙2 : 처음 만나는 점(.) 이후 부분은 제외한다. -> google

규칙3 : 남은 글자 중 처음 세자리 + 글자 갯수 + 랜덤한 세자리 자연수 + "!"로 구성한다. -> goo6812!

#### 예제 입력

```
https://naver.com
```

#### 예제 출력

```
nav5725!
```
